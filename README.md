# Fitbit Calorie Burn Prediction & Clustering

## Overview
This project predicts calories burned during workouts using machine learning and identifies workout patterns using clustering.

# Dataset
- 14,102 records
- Features: Age, Gender, Heart Rate, Workout details, etc.
- Target: Calories_Burned

# Models Used
- Linear Regression
- KNN Regressor
- Decision Tree
- Random Forest
- XGBoost

# Results
- Best R2 Score: 0.99
- Silhouette Score (Clustering): 0.38

# Key Insights
- Calorie burn is highly influenced by heart rate and MET values
- Clear workout intensity clusters identified (low, moderate, high)

# Tools & Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

# Conclusion
The project successfully combines regression and clustering to provide insights for fitness tracking applications.
